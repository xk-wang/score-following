Main file: batchProcessing.m

This code also requires the following toolboxes:
- nmflib: http://www.ee.columbia.edu/~grindlay/code.html
- crfChain: http://www.cs.ubc.ca/~murphyk/Software/CRF/crf.html
- timidity++: http://timidity.sourceforge.net/

For optimal reproduction, use the Merlin Vienna soundfonts for timidity: http://ocmnet.com/saxguru/Timidity.htm

Also required is the dataset from: http://c4dm.eecs.qmul.ac.uk/rdr/handle/123456789/13 (placed in the 'Dataset' folder)
